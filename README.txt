Adds a video column on the posts admin page for easier track which posts you've already made video for.

== Description ==

This plugin simply adds a new "video" column to the posts admin screen for you to easily track which posts you already have made a video for and which haven't. Each row contains a checkbox corresponding to each post. The checkbox status (checked or unchecked) is automatically saved on the fly.

== Installation ==

You can either:

1. Upload unzipped `video-checkmark` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin `video-checkmark` through the 'Plugins' menu in WordPress.
1. The new `Video` column can be found in the All Posts page of WordPress admin.
1. There is no option to configure.

Or:

1. Search `Video Checkmark` in the Add New plugin section in WordPress admin and install it.
1. The new `Video` column can be found in the All Posts page of WordPress admin.
1. There is no option to configure.
